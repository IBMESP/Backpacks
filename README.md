### Commands
- bp create <s/m>
- bp open
- bp delete
- bpsee <player>
- bgamerule keepBackpack <true/false> or bgr keepBackpack <true/false>
  
### Permisions
- bp.small (for /bp create s)
- bp.medium (for /bp create m)
- bp.bpsee (for /bpsee)
- bp.gamerule (for /bgamerule keepBackpack)
